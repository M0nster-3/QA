# 红线：ai_keywords 写入检查

Write 前逐条自查：

- [ ] 只更新了 ai_keywords 字段，其余字段一字未动。为什么：其余字段由数据获取脚本写入，覆盖会导致元数据丢失
- [ ] 关键词数量 5-15 个
- [ ] 每个 keyword 是数学概念级别（非单词、非句子）。为什么：单词级无区分力，句子级无法检索
- [ ] rank 从 1 起始，连续递增不跳号
- [ ] rank 按核心程度排序，主定理相关排前。为什么：rank 用于前端排序和搜索权重，乱序误导用户
- [ ] 无重复关键词（同概念不同表述已合并）
- [ ] 语言与论文一致。为什么：翻译引入术语歧义
- [ ] 未修改 title、authors、abstract 等其他字段

# 红线：paper.md 写入检查

Write 前逐条自查：

- [ ] 所有数学公式用 `$...$`（行内）或 `$$...$$`（独立行）包裹，LaTeX 语法正确
- [ ] PDF 中的黑板粗体（ℝ, ℕ, ℂ）对应 `$\mathbb{R}$`, `$\mathbb{N}$`, `$\mathbb{C}$` 等，不是随意猜测。为什么：黑板粗体在数学中有特定语义，猜错会改变数学含义
- [ ] 定理/定义/引理/命题/推论/注记 全部用 `**Type Number.**` 格式标记。为什么：M0 Agent 靠确切的 `**` 标记识别结构边界，格式偏差会导致漏切或错切
- [ ] 证明块用 `*Proof.* ...` 格式
- [ ] 章节标题用 `##` 或 `###` 标记
- [ ] 引用编号与 PDF 原文一致
- [ ] 文末有 `## References` 段，每条含作者、标题、期刊/出版社、年份
- [ ] 所有数学内容（定义、命题、证明）保留完整，无丢失
- [ ] 数学符号和含义未改变
- [ ] 论文全文完整转录，无内容丢失（动机叙述、历史背景、数学内容全部保留）
